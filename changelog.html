  • Layer-based linguistic exploration (ADR-005)

November 2025

  • Python to Rust migration (ADR-001)
    • Complete backend rewrite in Rust
    • 10-100x performance improvement (10-50ms search vs 500-1000ms)
    • Memory reduced from 200MB+ to ~50MB
    • Single static binary deployment
  • SQLite for storage (ADR-002)
    • Chose SQLite with sqlx for async queries
    • Zero configuration, single file database
    • Sub-millisecond query latency for indexed lookups
  • Tantivy for search (ADR-003)
    • Pure Rust full-text search engine
    • Sub-50ms queries for morphological searches
    • ~8MB index size for full Quran corpus
    • Faceted search on root, POS, pattern, verb_form, gender, number, case, voice, mood, tense, aspect
  • Vanilla JavaScript frontend (ADR-004)
    • No frontend framework (282 LOC total)
    • Zero npm packages for runtime
    • Terminal-like interface with Arabic RTL support
    • Built with Tauri for native desktop packaging
  • Tauri desktop app archived
    • Shifted to OpenCode + MCP server integration
    • Desktop Tauri app moved to archive/desktop-tauri-20260120/
    • Rust engine moved to archive/engine-rust-20260120/
