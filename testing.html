Testing Strategy

Testing approach across Syntra projects: unit tests, contract tests, and integration tests.

Unit Tests

Pure functions with no side effects. Test parsing, analysis, and mathematical computations.

Contract Tests

API endpoints validated against expected response shapes and invariants.

Integration Tests

End-to-end workflows testing full pipelines from input to output.

Projects

Scribe

Audio transcription server. Unit tests cover:

  • File saving and directory management
  • Job state transitions (running, done, error)
  • CORS handling

Run with: python -m pytest tests/

Linguist

Quranic analysis server. Tests cover:

  • Database operations (claims, evidence, patterns)
  • MCP tool invocations
  • Query validation

Test Data

Use small, deterministic fixture datasets to keep tests fast and reproducible.
